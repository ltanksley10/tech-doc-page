Intro to Loops

One very common thing you will find in JavaScript is a loop. Loops are used for condition based tasks when you may need to loop through them multiple times to find the condition.

As long as the condition is true, a loop will continue to run. Listed below are the most common forms of loops you will see.

For Loop

These types of loops are commonly used to iterate over an array multiple times. For loops consist of three expressions and a statement:

  • initialization: this is run before the first execution of the loop and is typically used to create counters
  • condition: an expression that gets checked prior to the execution of each iteration(checking whether true or false)
  • final-expression: the expression that gets run after each iteration, usually used to increment or decrement a counter
  • statement: the code to be repeated in the loop

Any of these three expressions or the statement can be ommitted if needed.

Example: for (let i = 0; i < arr.length; i++) { //statement };

To avoid exceeding the bounds of the array when indexing over it either set the condition to i < arr.length or i <= arr.length - 1.

For in Loop

This loop statement iterates over enumerable properties of an object in random order.

Example: for (var a in b) { //statement };
For of Loop

This statement creates a loop that iterates over iterable objects, which invokes a custom hook with statements to be executed for the value of each property.

Example: for (var of object) { //statement };

Iterable objects:

  • Arrays
  • Map
  • Set
  • Arguments
  • etc..
While Loop

The while loop first evaluates the condition. If the condition is true the statements are then executed. If the condition is false the statements are not executed and the loop ends.

Example: while(condition) { //statement(s) };
Do while Loop

The do while loop is similar to the while loop. However, in the do while loop the condition is checked at the end of the loop.

Example: do { //statement(s); } while (condition);

How to iterate with Do..While Loops:

var myArray = []; var i = 10; do { myArray.push(i); i++; } while(i <= 10);
